## 일지
### 2020-10-22
#### 기획

**메인 화면**

- 현재 시스템의 날짜와 시간을 출력합니다.
- 현재 시스템의 시간 정보에 맞게 아스키 아트를 출력합니다.
- "`로그인 한 멤버명`님, 총 `완료하지 않은 Todo 개수` 개의 할 일이 남았습니다." 메시지를 출력한다.
- 시스템 날짜에 따른 할 일 정보 목록을 출력한다.
 - 시간 순으로 정렬한다.
 - 할 일 정보: 완료여부(● : 완료, ○ : 미완료), 항목, 제목
